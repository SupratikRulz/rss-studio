# RSS Studio
RSS Studio is a RSS Reader app which allows user to subscribe, search for RSS, bookmark RSS articles and organise the feeds sources in folder.

This is a next js app which is mobile responsive.

Use frontend-pattern skills for architecting the app structure and components. This is used for best practices in react.

Use tailwind-design-system skills for creating the design tokens and components.

Use web-animation-design skills for implementing animations.

## Important
Donot use external component library like shadcn, radix ui etc. All the components should be made from scratch.
